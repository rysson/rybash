RyBash lib
==========

Small BASH library.


Install
-------

Clone the project and install (add source to .bashrc).

```bash
mkdir -p "$HOME/.local.share"
cd "$HOME/.local.share"
https://git.sharkbits.com/misc/shell/rybash.git
./rybash/install.sh
```

Relogin.


Usage
-----

```bash
rybash-include MODULE
```

Where `MODULE` is requested module name.
